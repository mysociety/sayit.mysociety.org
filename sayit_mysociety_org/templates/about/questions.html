{% trans "What is SayIt?" %} #

{% trans "It’s an open source tool for making transcripts easy to read, search and share on the modern internet." %}

{% trans "What is a Poplus Component?" %} #

{% blocktrans %}Poplus is a global collaboration of groups that believe it is currently too difficult and expensive to build effective new digital tools to help citizens exert power over institutions. Poplus Components are loosely joined tools, mostly structured as web services, that can be used to radically decrease the development time of empowerment sites and apps.{% endblocktrans %}

{% trans "Why try to make transcripts better?" %} #

{% blocktrans %}Transcripts are a kind of oil that greases the wheels of well-functioning societies. They let people discover when powerful people have made pronouncements that affect less powerful people. We believe that by making transcripts function better, more people will end up learning about decisions and opinions that affect their lives.{% endblocktrans %}

{% trans "What sort of people or organisations can SayIt help?" %} #

{% trans "Lots, including:" %}

  • {% trans "Journalists, who want to store, search and publish both the transcripts of specific events, and databases of speeches and statements by politicians." %}
  • {% trans "Officials in courts, who want to produce transcripts that are quicker and easier to use." %}
  • {% trans "Official in parliaments and other assemblies, who want to make transcripts that the public and politicians can more easily make use of." %}
  • {% trans "Researchers who have records of interviews with subjects, who want that data to be quick and easy to access, even if private." %}
  • {% trans "Campaigners and activists who want to track what certain people say about key issues." %}
{% trans "Why is a SayIt transcript better than a PDF version?" %} #

{% trans "SayIt has several advantages over classic paper or PDF versions:" %}

  • {% trans "The design is mobile responsive to make reading and navigation pleasant across devices, large and small" %}
  • {% trans "Search within an entire database of speeches, for example a whole trial, or everything said by an elected assembly – better than opening hundreds of PDFs and searching manually." %}
  • {% trans "Search within one person’s speeches, even if they only speak occasionally; and each speech is linked to a person, making it easy to see everything someone has said." %}
  • {% trans "Easily share links to a single session (for example at a trial), or even a single speech within a session." %}
  • {% trans "Make your transcripts more likely to show up clearly in search engines." %}
  • {% trans "There’s an API for people who want to do more advanced text processing." %}
{% trans "How do I get a transcript online using SayIt?" %} #

{% blocktrans %}You could convert your data to the right format, and work with us to get it uploaded to an instance here on sayit.mysociety.org. You could log in and author transcripts directly through this web interface, sharing your instance with others to help. Or you could install SayIt on your own server in your own project. More details.{% endblocktrans %}

{% trans "Why do you call this a Beta?" %} #

{% blocktrans %}In web development, ‘Beta’ is the name for a release of a piece of software which is in progress. In the case of SayIt, we are launching early so that we can learn more about what features people want us to build, and which are less important.{% endblocktrans %}

{% trans "Aren’t transcripts sort of old fashioned? Isn’t everything video now?" %} #

{% blocktrans %}Online video is definitely amazing stuff. But text transcripts offer certain virtues too, which makes them valuable in different ways. If you want to know whether a committee has been talking about you, or you want to subscribe to be sent alerts when your street gets mentioned by powerful people, then written transcripts, made of nicely structured data, are hard to beat. The ideal system would clearly mix written words, audio and video, and being open source we hope that SayIt can be integrated with other systems that specialise in AV. We’re keen to talk to people running projects in this field.{% endblocktrans %}

{% trans "Is SayIt open source? Does it use open standards or open data?" %} #

{% blocktrans %}SayIt is open source, licensed under the Affero GPL, and you can check out the code in our GitHub repo. The data standard we use is a subset of the Akoma Ntoso data standard – a standard developed specifically for use in parliaments and assemblies.{% endblocktrans %}

{% trans "Can I write a parser that loads data directly into SayIt on a daily or weekly basis?" %} #

{% trans "Yes – we’re keen to see people use the site like this. Please contact us so we can discuss how best to implement this so it works well for you and us." %}

{% trans "Does SayIt support languages other than English?" %} #

{% blocktrans %}Yes. Transcripts can be authored in any language, and we welcome translations of the interface at our project on Transifex. Do contact us if you want to help us launch a non-English SayIt site.{% endblocktrans %}

{% trans "Can SayIt sites be made private?" %} #

{% blocktrans %}Yes, please contact us if you want to discuss this feature. We understand that some organisations, for example newspapers, might want to run private databases of speeches for business purposes.{% endblocktrans %}

{% trans "Can SayIt be used to track speeches made in more than one place?" %} #

{% blocktrans %}Yes, you don’t just have to record one event, like a trial or a focus group. You can build up nicely structured database of quotes made by people across media sources, or from books and papers. We would like to see people use SayIt to build sites like ‘all public statements by the Prime Minister’.{% endblocktrans %}
